# AiAgent-Java

Enterprise AI Agent Framework for Java.

## Purpose

AiAgent-Java is a production-ready, enterprise-grade AI Agent Framework built on Java 21 and Spring Boot 3.5.x.

## Positioning

- **Enterprise AI Agent Framework**
- **Java Equivalent of LangGraph**
- **Java Equivalent of Spring AI**
- **Java Equivalent of OpenAI Agent SDK**

## Core Principles

| Principle | Description |
|-----------|-------------|
| AI Native | Built from the ground up for AI workloads |
| Documentation First | Every design decision is documented |
| Architecture First | Architecture precedes code |
| Convention Over Configuration | Sensible defaults |

## Technology Stack

| Layer | Technology |
|-------|-----------|
| Language | Java 21 |
| Framework | Spring Boot 3.5.x |
| Build | Maven |
| Database | PostgreSQL, Redis |
| LLM | Ollama, Qwen3-VL |
