# AI Operating System (AIOS)

## Purpose

AIOS is the governing specification of the AiAgent-Java project. It defines the project's constitution.

**Every contributor (human or AI) must read and follow AIOS before making any changes.**

## Scope

AIOS applies to all aspects of the AiAgent-Java project.

## Design Principles

| Principle | Description |
|-----------|-------------|
| Constitution First | AIOS is the supreme authority |
| Universal | All AI tools must read AIOS |
| Living Document | AIOS evolves with the project |
| Self-Documenting | Every section explains its purpose |

## How to Use AIOS

### For AI Agents

1. **Before any task**: Read the relevant AIOS sections
2. **Before writing code**: Read `02_RULES/` and `01_ARCHITECTURE/`
3. **Before creating agents**: Read `03_AGENT/`
4. **Before committing**: Read `02_RULES/COMMIT_RULES.md`
