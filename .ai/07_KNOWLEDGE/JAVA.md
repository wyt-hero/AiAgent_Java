# Java Knowledge Base

## Purpose
Reference for Java 21 features used in AiAgent-Java.

## Key Features
- Virtual Threads: `Executors.newVirtualThreadPerTaskExecutor()`
- Records: Immutable data carriers
- Sealed Classes: Finite state sets
- Pattern Matching: `instanceof` + switch patterns
- Text Blocks: Multi-line strings

## Best Practices
- Records for data carriers
- Sealed interfaces for state machines
- Virtual threads for I/O-bound execution
