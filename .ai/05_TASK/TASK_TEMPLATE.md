# Task Template

## Purpose
Standard template for development tasks.

## Required Fields
- Task ID (TASK-XXXX), Milestone, Priority (P0-P3), Status (TODO/IN_PROGRESS/DONE/BLOCKED)
- Objective, Background, Preconditions
- Scope (In/Out), Deliverables, Acceptance Criteria
- Forbidden section, Commit Message, Next Task

## Forbidden
- Starting without a task document
- Tasks without acceptance criteria
