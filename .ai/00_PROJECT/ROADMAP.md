# Project Roadmap

## Purpose

Defines the AIOS-level roadmap for project evolution.

## Scope

Covers AIOS specification evolution and framework development phases.

## Design Principles

- Milestones are sequential
- Each milestone has clear entry and exit criteria
- Specification changes are versioned independently

---

## 1. AIOS Specification Roadmap

### v1.0.0 — Initial Constitution (Current)

## 2. Framework Release Roadmap

| Version | Milestone | Target | Key Deliverables |
|---------|-----------|--------|------------------|
| 0.1.0 | Foundation | 2026-Q3 | Project structure, AIOS, Maven modules |
| 0.2.0 | Agent Kernel | 2026-Q3 | Core execution loop, prompt pipeline |

## Forbidden

- Starting work on a milestone before the previous one is complete
