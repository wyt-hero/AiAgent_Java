# Project Overview

## Purpose

This document defines the AiAgent-Java project identity, vision, scope, and governing principles.

## Scope

Applies to every contributor, AI agent, and human developer working on this project.

## Design Principles

| Principle | Description |
|-----------|-------------|
| AI Native | Architecture designed for AI workloads from day one |
| Documentation First | No code without prior documentation |
| Architecture First | Architecture decisions precede implementation |
| Convention Over Configuration | Sensible defaults |
| Enterprise Ready | Built for production, scalability, and maintainability |

---

| Attribute | Value |

| License | Apache 2.0 |

## 2. Vision

AiAgent-Java aims to be the **definitive enterprise-grade AI Agent Framework for the Java ecosystem**.

## 3. Positioning

| Comparison | Description |
|------------|-------------|
| Java LangGraph | Graph-based agent orchestration |
| Java Spring AI | Deep Spring ecosystem integration |
| Java OpenAI Agent SDK | Multi-agent collaboration |

## 4. Core Capabilities

- Agent Kernel with think-act-observe loop
- Prompt Pipeline with template engine
- Multi-layer Memory System
- Graph-based Workflow Engine
- Tool Registry with MCP compliance
- Configurable Persona System

- Skipping documentation
